## Data:

The data folder for this repository was put into the .gitignore for storage purposes.

How to access the data:

-   **UFO Sightings:** This dataset contains reported UFO sightings across the US. It can be accessed via the TidyTuesday GitHub repository at: <https://raw.githubusercontent.com/rfordatascience/tidytuesday/main/data/2023/2023-06-20/ufo_sightings.csv>

-   **Places:** This dataset contains population and location data for US places. It can be accessed via the TidyTuesday GitHub repository at: <https://raw.githubusercontent.com/rfordatascience/tidytuesday/main/data/2023/2023-06-20/places.csv>
